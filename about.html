このサイトについて

Google Sites からリンクする、無料の本文サイトとして使う前提で作っています。

目的

  • 静的 HTML / CSS だけで運用する。
  • 白背景で、スマホでも読みやすくする。
  • 記事をカテゴリごとに増やしやすくする。

カテゴリ構成

  • 模糊コラム: テーマ単位の束、小片、断章。
  • AI評論: AI、道具、表現、創作、開発の評論。
  • 社会分析: 行政、制度、労働、世代、地域、構造の分析。
  • メモ・記録: 作業メモ、運用メモ、試行錯誤、更新記録。

補助ページ

公開サイトには 自作Python配布 の入口も置き、ZIP 本体は GitHub Releases 側で配布します。

記事追加方法

  1. articles/ に新しい HTML を 1 つ追加する。
  2. その記事をカテゴリページの「最新3件」に入れる。
  3. 必要ならトップページの 3 件リンクも更新する。

トップページの最新3件リンクを更新する手順

  1. 各カテゴリで最新記事の順番を確認する。
  2. index.html の各カード内リンクを差し替える。
  3. カテゴリページ側の「最新3件」も同じ順にそろえる。

このサイトについて

このサイトは、Google Sites からリンクして使う「本文の置き場」を想定して作っています。 余計な機能を入れず、読むことと追加することを優先しています。

方針

  • 静的 HTML と CSS だけで構成する。
  • 白背景と十分な余白で、スマホでも読みやすくする。
  • 記事をカテゴリに分け、一覧から辿りやすくする。
  • 今後は 1 記事 1 ファイルで増やせるようにする。

運用の考え方

新しい記事を増やすときは、まずカテゴリページにリンクを置き、 必要なら文章一覧にも追加します。これで、トップページを大きく変えなくても サイト全体を整理できます。

公開先

GitHub Pages のルートにこのフォルダを置けば、そのまま公開できます。 画像や JavaScript に依存していないため、更新も軽く済みます。
